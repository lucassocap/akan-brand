Animation System (Hims + Leonardo.ai inspired)

Animation Duration Easing CSS / Framer Motion Where Used Inspiration
SlideUp 600ms cubic-bezier(0.16,1,0.3,1) opacity 0→1, y 24→0 Default for all sections, paragraphs, headings Hims — clean section reveals
SlideLeft 600ms cubic-bezier(0.16,1,0.3,1) opacity 0→1, x -40→0 Hero text, left-side content in split layouts Hims — hero text entrance
SlideRight 600ms cubic-bezier(0.16,1,0.3,1) opacity 0→1, x 40→0 Hero image, right-side content in split layouts Hims — image entrance
ScaleIn 300ms cubic-bezier(0.34,1.56,0.64,1) opacity 0→1, scale 0.92→1 Quiz container, badges, toasts, modals Leonardo.ai — card pop-in
StaggerChildren 500ms per child ease-out staggerChildren: 0.1 Trust bar, steps, product grid, testimonials, triage cards Hims — product grid reveal
TextReveal 600ms per word cubic-bezier(0.16,1,0.3,1) opacity 0→1, y 20→0, blur 4→0 Hero headline only (word-by-word) Leonardo.ai — text generation feel
HoverLift 200ms ease-out translateY(-2px) + shadow All interactive cards Hims — product cards
ImageZoom 500ms cubic-bezier(0.16,1,0.3,1) scale(1.05) on hover Product card images, hero image Hims — rollover image zoom
SubtlePulse 3000ms loop ease-in-out scale 1→1.03→1 Primary CTA button (idle state only) Leonardo.ai — CTA attention
ProgressFill 800ms ease-out width 0→N% Quiz progress bar Hims — quiz flow
NavBlur 300ms ease backdrop-filter: blur(16px) Fixed navigation bar Leonardo.ai — frosted glass nav

Implementation Notes for AI Builder:

  • React: Use motion (framer-motion successor) for all animations. whileInView for scroll triggers.
  • Trigger: IntersectionObserver with threshold: 0.2 and once: true.
  • Reduced motion: Wrap all animations in @media (prefers-reduced-motion: no-preference).
  • Hero text reveal: Split headline into <span> per word, use staggerChildren: 0.08.
  • Stagger pattern: Parent gets staggerChildren: 0.1, children get variants={{ hidden: ..., visible: ... }}.
  • Nav scroll: Add .scrolled class after 50px scroll with requestAnimationFrame.
  • Horizontal scroll: Testimonials use native CSS scroll-snap, no JS library needed.
  • No Aceternity UI needed for v1: Pure framer-motion covers all effects. Aceternity optional for hero spotlight later.
